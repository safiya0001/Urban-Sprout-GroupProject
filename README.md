# Urban Sprout Group Project

Urban Sprout is a static multi-page website created for the Information System Spring Intake Group C8. It presents a gardening-focused storefront with product listings, blog content, research pages, team profiles, and a contact form.

## Project Overview

The site is designed around an urban gardening theme and includes:

- A homepage with featured categories and product highlights
- A product section for gardening items and supplies
- A blog section with multiple articles
- A research page for supporting project content
- An about-us area with team member portfolios
- A contact page for enquiries and feedback

## Folder Structure

- `index.html` - homepage
- `About-Us/` - about page and team portfolios
- `Blog/` - blog landing page and blog posts
- `Contact-Us/` - contact form page
- `Products/` - product pages and category folders
- `Research/` - research page
- `styles/` - shared stylesheet
- `images/` - site assets

## How To Run

This project does not require any build step or package installation. To view it locally:

1. Open the folder in VS Code or your file explorer.
2. Open `index.html` in a browser.
3. Optionally use VS Code Live Server for a smoother preview while editing.

## Notes

- The project is built with plain HTML and CSS.
- Internal page links are used to navigate between sections of the site.
- Some product and blog pages are separated into individual HTML files for easier organization.
